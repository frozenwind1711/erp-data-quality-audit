# ERP Data Quality Audit — AdventureWorks

## Overview
A SQL-based data quality audit framework applied to AdventureWorks ERP dataset,
simulating real-world data validation scenarios in NetSuite/Oracle environments.

## Audit Areas
| # | Area | Description |
|---|------|-------------|
| 1 | NULL Audit | Identify critical columns with unexpected NULL values |
| 2 | Duplicate Audit | Detect duplicate records by business key |
| 3 | Anomaly Audit | Flag outliers — negative values, invalid dates, price anomalies |
| 4 | Referential Integrity | Validate FK relationships across Sales, HR, Production modules |

## Tech Stack
- Oracle SQL
- AdventureWorks schema (OLTP)
